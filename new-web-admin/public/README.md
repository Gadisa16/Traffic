# Public Assets

This directory contains static assets served by the application.

## Files

- **favicon.ico**: Browser tab icon (replace with TMS logo)
- **og-image.png**: Social media preview image (1200x630px recommended)
  - Currently a placeholder - replace with professional TMS branding image
  - Should include: TMS logo, "Traffic Management System", "Sidama Region Transport Authority"
- **placeholder.svg**: Generic placeholder for missing images
- **robots.txt**: Search engine crawler instructions

## Updating OG Image

To create a professional OG image:

1. Design a 1200x630px image with:
   - TMS logo/branding
   - System name: "Traffic Management System"
   - Authority: "Sidama Region Transport Authority"
   - Professional color scheme matching the app theme

2. Replace `og-image.png` with your designed image

3. The image will be used for:
   - Social media link previews (Twitter, Facebook, LinkedIn)
   - Search engine results
   - Messaging apps link previews
